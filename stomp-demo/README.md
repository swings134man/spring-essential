## STOMP-DEMO Project 

- STOMP + WebSocket 을 사용한 실시간 채팅 데모
- n개의 채팅방, n명의 사용자 지원
- Spring Boot + Spring WebSocket + STOMP 을 사용하여 구현
- 별도의 MQ 브로커 없이, Spring Boot 내장 WebSocket 서버로 구현(in-memory) - 스프링에서 제공하는 기능들로만 구현된 간단한 데모 프로그램

### 주요 기능
- 사용자는 1번의 1개의 채팅방에 입장하여 메시지를 주고받을 수 있음
- 사용자들은 채팅방에 입장하여 메시지를 주고받을 수 있음
- 사용자는 메인페이지에서 채팅방 개설, 채팅방 목록 조회, 채팅방 입장 가능
- 채팅방 입장 시, 해당 채팅방의 메시지 구독
- 별도의 인증 절차 없이, 사용자 이름과 채팅방 이름을 입력하여 입장 가능 시스템
- 채팅 메시지는 JSON 형식으로 전송되며, 메시지에는 발신자 이름, 메시지 내용, 타임스탬프 등이 포함됨

- 프론트의 경우 Thymeleaf 템플릿 엔진을 사용하여 HTML 페이지를 렌더링하며, JavaScript를 사용하여 WebSocket 연결 및 메시지 송수신을 처리함
- 디자인에 중심은 두지말것(심플하게 구현), 기능 구현에 집중 및 코드의 가독성, 유지보수성에 집중할 것
